Reinforcement Learning Course Project ‚Äî CartPole Swing-Up with PPO

In this course project, we applied Proximal Policy Optimization (PPO) to solve the CartPole Swing-Up task, learning to swing the pole from a downward configuration to upright stabilization. To improve data efficiency and training stability, we collected rollouts from eight parallel environments with randomized initial conditions and performed updates using mini-batch SGD. The implementation includes a hand-built policy network for action selection and a value network for state-value estimation during PPO training.
